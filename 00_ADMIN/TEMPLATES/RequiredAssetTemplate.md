This document defines the mandatory fields for all NASA-required models, including:
CTBs, Stowage Enclosures, Payload Banks, Freezers, and Freezer Bags.

---

## Documentation Requirements

### 1. Asset Name
Name of the NASA-required asset.

### 2. Author and School
Full name(s) of the student(s) and the school name.

### 3. Date
Date the model documentation was completed.

### 4. Description
A brief description of the function and purpose of the required asset.

### 5. Real-World Dimensions
Include key measurements taken from Blender in real-world units.
Use a table when possible.

### 6. Files Included
List all files included with the asset:
- low-poly .fbx  
- high-poly .fbx  
- .blend file  
- textures folder  
- preview.png  

### 7. Preview Image
A small render or screenshot of the asset.

### 8. Version History
A simple log of changes, starting with the first release.
