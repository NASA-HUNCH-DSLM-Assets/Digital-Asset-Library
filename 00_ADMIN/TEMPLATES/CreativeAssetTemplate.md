This document defines the minimum requirements for all creative, conceptual, or original models made by students.

---

## Documentation Requirements

### 1. Model Name
Name of the creative or conceptual model.

### 2. Author and School
Full name(s) of the student(s) and the school name.

### 3. Date
Date the model documentation was completed.

### 4. Description
A short explanation summarizing what the model is and its purpose.

### 5. Files Included
List all files included with the asset:
- .fbx file  
- preview.png  

### 6. Preview Image
A small render or screenshot of the asset.

### 7. Version History
A simple log of changes, starting with the first release.
