# NASA Required Assets List  
### Deep Space Logistics Module (DSLM) — NASA HUNCH

This document lists all mandatory assets requested by NASA’s Deep Space Logistics Module (DSLM) and HUNCH program.  
These assets must be modeled accurately and documented according to NASA specifications.

---

## 1. Cargo Transfer Bags (CTBs)

NASA requires CTB sizes from 0.5 through 10.0.

Required CTB models:

- CTB_0.5
- CTB_1.0
- CTB_1.5
- CTB_2.0
- CTB_2.5
- CTB_3.0
- CTB_4.0
- CTB_5.0
- CTB_6.0
- CTB_7.0
- CTB_8.0
- CTB_9.0
- CTB_10.0

Each size should include:
- Preview image  
- README documentation  

---

## 2. Stowage Enclosures

Required models:
- Single stowage enclosure  
- Double stowage enclosure  
- Triple stowage enclosure  
- Quad stowage enclosure  

---

## 3. Payload Banks

Required:
- Payload Bank (holds 4 enclosures)  
- Payload Bank (holds 8 enclosures)

These serve as mounting plates for stowage enclosures.

---

## 4. Moon-to-Mars Freezer

Required model:
- Moon-to-Mars Freezer

---

## 5. Moon-to-Mars Custom Freezer Bag

Required model:
- Custom freezer bag designed for Moon-to-Mars program

---

## Notes

- All assets must use correct real-world dimensions.  
- Each asset must include a README, preview image, and source files.  
- Document thoroughly and maintain version history.
