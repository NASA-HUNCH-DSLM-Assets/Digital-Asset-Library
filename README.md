# NASA HUNCH Digital Asset Collaboration Project (Digital Asset Library)

This repository is the central library for NASA HUNCH Digital Assets (3D models, textures, and related files) supporting VR/AR and simulation training.

## Quick Start
1. Read `00_ADMIN/PROJECT_GUIDE.md`
2. Follow `00_ADMIN/WORKFLOW.md` for the submission and review process
3. Use the standardized README templates in `00_ADMIN/TEMPLATES/` for every asset folder

## Repo Map
- `00_ADMIN/` Project rules, templates, workflows, onboarding materials
- `01_REFERENCE/` NASA specs, dimension tables, and NASA-provided reference models
- `02_REQUIRED_MODELS/` NASA-requested required models (owned by the ARTIST team / maintainers)
- `03_SCHOOL_SUBMISSIONS/` Per-school folders for drafts and submissions (pull requests target this area)
- `04_SHARED_LIBRARY/` Approved, reusable assets for all schools
- `05_ARCHIVE/` Deprecated or replaced assets

## Large Files
This repository uses Git LFS for large binary assets such as `.fbx`, `.blend`, textures, and archives.
See `.gitattributes`.
