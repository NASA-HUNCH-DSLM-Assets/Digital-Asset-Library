# Contributing

## Where to submit
All school submissions must be added under:
`03_SCHOOL_SUBMISSIONS/SXX_SchoolName/`

Do not directly edit `02_REQUIRED_MODELS/` unless you are part of the maintainers/ARTIST team.

## Pull Request rules
- One asset per PR whenever possible
- Include preview image(s)
- Include a complete README using the templates in `00_ADMIN/TEMPLATES/`
- Use Git LFS for large files

## Review process
A reviewer will request changes or approve. Maintainers publish approved assets to `04_SHARED_LIBRARY/`.

## Questions
Use GitHub Issues for questions, bugs, and requests for new assets.
